# Exercices de Programmation Orientée Objet en Java

Ce dossier contient tous les exercices pratiques pour apprendre la POO en Java.

## Structure des exercices

### Exercice 1.1, 1.2, 1.3 : Projet `automobile`
- **Voiture.java** : Classe représentant une voiture avec :
  - Attributs privés (marque, modèle, couleur, numéro de châssis, nombre de places)
  - Trois constructeurs (par défaut, d'initialisation, de copie)
  - Méthode `afficher()` pour afficher les informations
  - Accesseurs (getters et setters) pour tous les attributs
  - Méthode `estEgal()` pour comparer deux voitures
  
- **Automobile.java** : Classe principale pour tester la classe Voiture

### Exercice 1.5 : Projet `programme`
- **Personne.java** : Classe représentant une personne avec un nom
- **Programme.java** : Classe principale pour comprendre la notion de référence

### Exercice 1.6 : Projet `equation`
- **EquationSecondDegre.java** : Classe pour résoudre les équations du second degré
- **TestEquation.java** : Classe pour tester toutes les méthodes

### Exercice 1.7 : Projet `complexe`
- **Complexe.java** : Classe pour représenter et manipuler les nombres complexes
- **TestComplexe.java** : Classe pour tester toutes les opérations sur les complexes

### Exercice 1.8 : Projet `bibliotheque`
- **Livre.java** : Classe pour représenter un livre dans une bibliothèque
- **TestBibliotheque.java** : Classe pour tester toutes les fonctionnalités

## Comment utiliser ces exercices

1. Ouvrez chaque projet dans votre IDE Java
2. Compilez les fichiers .java
3. Exécutez les classes avec la méthode `main()` pour voir les résultats
4. Lisez les commentaires dans le code pour comprendre chaque étape

## Concepts abordés

- Création de classes et d'objets
- Constructeurs (par défaut, d'initialisation, de copie)
- Attributs privés et accesseurs (getters/setters)
- Comparaison d'objets (== vs méthode personnalisée)
- Notion de référence en Java
- Méthodes privées et publiques
- Modélisation de problèmes réels avec des classes
